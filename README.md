# baf_RailSim2-Plugins
RailSim2用プラグインの保管兼配布用リポジトリ

## 保管されているもの
- `common`フォルダ
- テンプレート
- 芝野鉄道18000系

- 芝野鉄道13000系(old)
- 芝野鉄道Si2110系(old)
- `common_shib`フォルダ(old)

## 注意事項
**このリポジトリの内容を二次配布しないでください。**
<br />
また、プラグインをダウンロードする際は[リリースページ](https://github.com/baf1n/baf_RailSim2-Plugins/releases "リリースページ")からダウンロードしてください。
